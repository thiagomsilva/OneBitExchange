# OneBitExchange

Aplicação desenvolvida para conversão de moedas integrada com a API `https://currencydatafeed.com`

# Desenvolvimento

- Ruby On Rails
- PostgreSQL
- Docker
- Docker Compose
- Yarn
- Testes: RSpec

# Deploy

- Codeship
- Heroku

# Aplicação disponível em:

https://onebitexchangeprod.herokuapp.com
